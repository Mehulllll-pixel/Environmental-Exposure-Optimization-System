# 🌍 AI Environmental Exposure Optimizer

An AI-powered web application that helps users minimize environmental exposure risks by providing intelligent recommendations based on air quality, user profile, and location.

---

## 🚀 Live Website

👉 **Access the live application here:**  
🔗 https://mehullprediction.netlify.app/

---

## ✨ Features

- 📍 Location-based exposure optimization  
- 🧠 AI-driven recommendations  
- 📊 Exposure analytics (PEVI score)  
- 🕒 Best time & safe window suggestions  
- 🧬 Personalized based on:
  - Age group
  - Health condition
  - Risk preference  
- 🗺️ Interactive map visualization  
- ⚡ Fast & responsive UI  

---

## 🏗️ Tech Stack

### Frontend
- React (Vite)
- CSS (Glassmorphism + animations)
- Leaflet (Maps)

### Backend
- FastAPI (Python)
- Pandas (Data processing)
- Custom optimization engine

### Deployment
- Frontend: Netlify  
- Backend: Render  

---

## ⚙️ How It Works

1. User selects:
   - Location  
   - Date  
   - Age group  
   - Health condition  
   - Risk preference  

2. Backend processes:
   - Environmental dataset  
   - Calculates PEVI score  
   - Runs optimization logic  

3. System outputs:
   - Best hour to travel  
   - Safe exposure window  
   - Risk level  
   - AI insights
